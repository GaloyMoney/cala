## Create a journal
### GraphQL body
```graphql
mutation journalCreate($input: JournalCreateInput!) {
  journalCreate(input: $input) {
    journal {
      journalId
      name
    }
  }
}
```
### Variables
```json
{
  "input": {
    "journalId": "bcc24f47-990c-457d-88cb-76332450ac77",
    "name": "General Ledger"
  }
}
```

### Response

```json
{
  "data": {
    "journalCreate": {
      "journal": {
        "journalId": "bcc24f47-990c-457d-88cb-76332450ac77",
        "name": "General Ledger"
      }
    }
  }
}
```
